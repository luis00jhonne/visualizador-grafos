Projeto de um Visualizador de Grafos utilizando Matriz de Adjacência.
É feito a leitura de um arquivo no formato abaixo:

```
ND
vertice1,vertice2
vertice1,vertice3
...
verticeX,verticeY
```

E obterá algumas informações sobre o Grafo.
